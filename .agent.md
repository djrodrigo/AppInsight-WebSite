# What AppInsight Does

AppInsight connects to Spring Boot applications through Spring Boot Actuator APIs and provides comprehensive operational visibility through a rich desktop dashboard.

The software visualizes:

* Application Health
* JVM Metrics
* Memory Usage
* CPU Usage
* Request Metrics
* Response Times
* Error Rates
* Thread Statistics
* Datasource Metrics
* HikariCP Pool Metrics
* Logger Configuration
* Environment Variables
* Application Information
* Custom Metrics

AppInsight helps engineering teams identify performance bottlenecks and operational issues quickly.
